* 카카오 클라우드 스쿨 클라우드 엔지니어 과정 수업에 대한 복습 스터디입니다.

* 평일 중 하루 발표시간 갖기?ㅎㅎ

## ✅ 참여 방법
1. 이 저장소를 `fork` 한다. 
2. 본인의 이름으로 디렉토리를 만든다.
3. 본인의 디렉토리 내에 markdown으로 정리한다.
4. 원본 저장소로 `Pull Request`를 한다. 

### 💡 예시
`hynjxn/openstack.md`
`hynjxn/ELK.md`
...

## ⚠️ commit 컨벤션
- commit 메세지: 복습한 내용에 대한 소제목
- 터미널에서 작성법:
```
git commit -m "openstack의 구성 요소, ..."
```
